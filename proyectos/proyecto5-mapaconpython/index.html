🗺️ Mapa Interactivo con Python y Folium

Visualización geográfica utilizando la librería Folium para Python.

📌 Herramienta utilizada

Este proyecto fue desarrollado con Python y la librería Folium, que permite renderizar mapas interactivos utilizando Leaflet.js.

⚙️ Código fuente


#generar un mapa

import folium

def generar_mapa():
    latitud = -34.4575
    longitud = -58.7061

    mapa = folium.Map(location=[latitud, longitud], zoom_start=13)

    folium.Marker(
        location = [latitud, longitud],
        popup="Benavidez",
        toolyip="Click para mas info",
    ).add_to(mapa)

    mapa.save("mapa.html")

generar_mapa()
